Security Market Indexes
Index Math · Weighting · Construction · Types · Uses
▾

Core Index Formulas

Price Return Index Value
V_PRI = (Σ nᵢ × Pᵢ) / Divisor
nᵢ = units held; Pᵢ = price; Divisor adjusted for splits, composition changes
Price Return (Single Period)
PR_I = (V_PRI₁ − V_PRI₀) / V_PRI₀ = Σ wᵢ × PR_i
wᵢ = weight of security i; PR_i = (P₁ᵢ − P₀ᵢ)/P₀ᵢ
Total Return (Single Period)
TR_I = (V_PRI₁ − V_PRI₀ + Inc_I) / V_PRI₀ = Σ wᵢ × TR_i
Inc_I = total dividends/income from all constituents; TR includes reinvestment of all income
Multi-Period Index Return
TR (n periods) = [(1+TR₁)×(1+TR₂)×...×(1+TRn)] − 1
Total Return Index ≥ Price Return Index always (due to reinvested income). They start equal at inception.

Index Weighting Methods

MethodFormula / BasisExample
Price-Weightedwᵢ = Pᵢ / ΣP; higher-priced stocks dominateDJIA, Nikkei 225
Market-Cap Weightedwᵢ = Mᵢ / ΣM; float-adjusted version commonS&P 500, MSCI
Float-Adjusted Mkt-CapExcludes non-trading shares (govt, insider)Most modern indexes
Equal-Weightedwᵢ = 1/N; requires frequent rebalancingValue Line
Fundamental-Weightedwᵢ based on earnings, dividends, sales, BVRAFI indexes
Price-weighted: stock splits distort weights (require divisor adjustment). Market-cap: momentum bias (overweight overvalued, underweight undervalued).

Index Construction Issues

  • Target Market – geographic, asset class, sector, strategy
  • Security Selection – objective or subjective criteria
  • Weighting – method chosen (above)
  • Rebalancing – restoring target weights; equal-weight needs frequent rebalancing; mkt-cap minimal
  • Reconstitution – adding/removing securities
Reconstitution causes: reconstitution effect – prices of added securities rise, removed fall, just before change date.

Uses of Security Market Indexes

  • Gauge market sentiment and performance
  • Benchmark for active portfolio managers
  • Proxy for systematic risk (market portfolio in CAPM)
  • Basis for index funds and ETFs (passive management)
  • Measure of market return for asset allocation
  • Basis for derivative instruments (index futures, options)

Market Efficiency
EMH · Three Forms · Anomalies · Behavioral Finance
▾

Efficient Market Hypothesis (EMH) – Core

  • Market Value – current trading price
  • Intrinsic Value (Fundamental Value) – true value based on all investment characteristics; discrepancy between market and intrinsic = profit opportunity
  • Informationally Efficient Market – prices reflect all available information quickly and rationally
  • Prices react only to unexpected/surprise information (not anticipated)
  • Consistent superior risk-adjusted returns impossible in efficient market
Abnormal Return
Abnormal Return = Actual Return − Expected Return
Expected return from CAPM or market model. Consistent abnormal returns = evidence against efficiency.

Three Forms of Market Efficiency (Fama)

FormReflectsImplication
Weak FormAll past market data (prices & volume)Technical analysis cannot earn abnormal returns
Semi-Strong FormAll public informationFundamental analysis cannot earn abnormal returns (net of costs)
Strong FormAll public AND private informationEven insiders cannot earn abnormal returns; NOT supported empirically
Semi-Strong ⊃ Weak form. Strong form ⊃ Semi-Strong ⊃ Weak form. Each form implies all weaker forms also hold.
Evidence: Developed markets ≈ Semi-strong efficient. Markets NOT strong-form efficient (insider trading laws exist because abnormal profits possible with private info).

Factors Affecting Market Efficiency

  • Number of Market Participants – more = more efficient
  • Information Availability & Disclosure – better info = more efficient
  • Limits to Arbitrage – trading costs, short-selling restrictions reduce efficiency
  • Transaction Costs – market efficient "within bounds of arbitrage"
  • Information-Acquisition Costs – Grossman-Stiglitz: return must cover research costs
  • Regulation FD (USA) – requires public disclosure of material non-public info given to professionals
Arbitrage: simultaneous buy/sell of identical assets in different markets at different prices → risk-free profit → drives prices together → enhances efficiency

Implications for Analysis & Portfolio Management

  • Fundamental Analysis – useful to help price discovery (facilitates semi-strong efficiency); can earn returns if comparative info advantage exists
  • Technical Analysis – cannot earn consistent abnormal profits in weak-form efficient market; however helps maintain weak-form efficiency by exploiting then eliminating patterns
  • Active Management – cannot consistently beat market; passive preferred (lower costs)
  • Event Study – test of semi-strong efficiency; measures abnormal returns around announcement dates
  • Role of portfolio manager: appropriate diversification, risk/return, tax considerations; NOT beating market

Overview of Equity Securities
Common & Preference Shares · Private Equity · DRs · ROE · Cost of Equity
▾

Common Shares (Ordinary Shares)

  • Ownership interest; residual claimant after debt and preferred
  • Voting Rights – elect board, approve major decisions
  • Statutory Voting – 1 share = 1 vote per director seat
  • Cumulative Voting – concentrate all votes on one candidate; better for minority shareholders
  • Proxy Voting – vote via designated party
  • Callable Common – issuer can repurchase at preset price
  • Putable Common – shareholder can sell back at preset price (less risky)
  • Dual-Class Shares – different voting rights (e.g., Class A voting, Class B non-voting)
Putable shares: lower risk → lower expected return. Callable shares: higher risk (capped upside) → higher dividend to compensate.

Preference Shares (Preferred Stock)

  • Priority over common shares in dividends AND liquidation
  • Generally NO voting rights
  • Fixed dividend (not contractually obligated but priority)
  • Hybrid: characteristics of both debt (fixed payments) and equity (no maturity, perpetual)
TypeFeature
CumulativeUnpaid dividends accumulate (must pay all before common dividends)
Non-CumulativeUnpaid dividends forfeited; still priority over common in current period
ParticipatingCan share in profits above stated dividend; extra distributions on liquidation
Non-ParticipatingFixed dividend only; par value on liquidation
ConvertibleCan convert to common at set ratio; common for VC/PE deals

Risk order: Debt < Preferred < Common (higher risk = higher expected return)

Return & Value Formulas

Total Return on Equity
Rt = (Pt − Pt−1 + Dt) / Pt−1
Pt = sale price; Pt−1 = purchase price; Dt = dividends received
Return on Equity (ROE)
ROE = Net Income / Average Book Value of Equity or ROE = Net Income / Beginning Book Value of Equity
Sustainable Growth Rate
g = b × ROE b = Retention Rate = (1 − Dividend Payout Ratio)
Market-to-Book (Price-to-Book) Ratio
P/B = Market Price per Share / Book Value per Share
BV per share = Total Shareholders' Equity / Shares Outstanding
Market Value of Equity (Market Cap)
Market Cap = Share Price × Shares Outstanding

Equity Value & Cost of Equity

  • Book Value of Equity = Total Assets − Total Liabilities; reflects historical decisions
  • Market Value of Equity = Market Cap; reflects future expectations
  • Intrinsic Value = PV of future cash flows; differs from both BV and MV
  • Cost of Equity = minimum return required by shareholders; estimated via DDM or CAPM
CAPM – Cost of Equity
r = Rf + β × (Rm − Rf) Rf = risk-free rate; β = beta; Rm = market return (Rm − Rf) = equity risk premium (ERP)
WACC
WACC = wd × rd × (1−t) + we × re wd = debt weight; rd = cost of debt; t = tax rate we = equity weight; re = cost of equity
If ROE > required return on equity → management creating value. If ROE < required return → destroying value. P/B > 1 implies growth expectations.

Company Research Report Structure

  • Front Matter – issuer name, symbol, recommendation, target price, disclosures
  • Recommendation – Buy/Hold/Sell + key reasons
  • Company Description – business model, revenue/profit disaggregation by product/geography
  • Industry Overview & Competitive Positioning – Porter's Five Forces, PESTLE, market shares
  • Financial Analysis & Model – historical + forecasted financials; key drivers
  • Valuation – target price; relative and PV approaches; scenario/sensitivity analysis
  • ESG Considerations – ownership structure, compensation, governance
  • Risks – material downside and upside risks
Initiation reports (new coverage) are comprehensive. Subsequent reports focus on updates after earnings releases or new information.

Revenue Analysis

  • Bottom-Up Approach – decompose revenue into price × volume by product/segment/geography
  • Top-Down Approach – revenue = market size × market share; link to GDP growth
  • Combine both approaches; cross-check assumptions
Revenue Decomposition (Bottom-Up)
Revenue = Volume × Average Selling Price (ASP) Or: Revenue = # Stores × Revenue per Store Or: Revenue = # Users × Revenue per User
Revenue Decomposition (Top-Down)
Revenue = Market Size × Market Share
Pricing Power – ability to raise prices without losing volume. Driven by market structure and competitive positioning. Key signal: rising margins over time.

Degree of Operating Leverage (DOL) & Degree of Financial Leverage (DFL)

Degree of Operating Leverage (DOL)
DOL = % Δ Operating Profit / % Δ Revenue (Sales) = (ΔOP/OP) / (ΔSales/Sales) Also: DOL = Q(P−VC) / [Q(P−VC) − FC] = Contribution Margin / Operating Profit
Higher fixed costs → higher DOL → more volatile operating profit for same % change in sales. DOL > 1 always (when FC > 0).
Degree of Financial Leverage (DFL)
DFL = % Δ Net Income / % Δ Operating Income = EBIT / (EBIT − Interest Expense) = Operating Income / (Operating Income − Int. Exp.)
Higher fixed financing costs (interest) → higher DFL. DFL close to 1 means low financial leverage. DFL × DOL = Degree of Total Leverage (DTL).
Degree of Total Leverage (DTL)
DTL = DOL × DFL = % Δ Net Income / % Δ Revenue

Working Capital Analysis

Cash Conversion Cycle (CCC)
CCC = DSO + DOH − DPO DSO = Days Sales Outstanding = AR / (Revenue/365) DOH = Days of Inventory on Hand = Inv / (CoGS/365) DPO = Days Payable Outstanding = AP / (CoGS/365)
Shorter CCC = less financing needed. Negative NWC = suppliers finance operations.
Net Working Capital (NWC)
NWC = Current Assets − Current Liabilities NWC/Sales ratio = capital intensity of working capital

Capital Investments & Capital Structure

ROIC (Return on Invested Capital)
ROIC = NOPAT / Invested Capital NOPAT = Net Operating Profit After Tax = EBIT × (1−t) Invested Capital = Total Assets − Non-interest-bearing Current Liabilities (or: Debt + Equity − Cash)
ROIC > WACC → economic value created (positive EVA)
DuPont ROE Decomposition
ROE = Net Margin × Asset Turnover × Equity Multiplier = (NI/Rev) × (Rev/Assets) × (Assets/Equity) Equity Multiplier = 1 / (1 − Debt/Assets) = 1 + D/E
Net Debt & Coverage
Net Debt = Total Debt − Cash & Investments Net Debt/EBITDA = leverage ratio Interest Coverage = EBIT / Interest Expense

Porter's Five Forces Framework (Determinants of Industry Profitability)

ForceKey Factors (Increases Threat)Reduces Profitability When...
1. Threat of New Entrants Low capital requirements, no economies of scale, no network effects, few regulatory barriers, low switching costs, no brand loyalty Easy entry → price competition → margins squeezed
2. Threat of Substitutes Lower-priced substitutes, improving substitute performance, low switching costs to substitute Caps pricing power for incumbents
3. Bargaining Power of Customers Few large customers, standardized/undifferentiated product, product not critical, large share of customer budget, ability to backward integrate Forces price concessions
4. Bargaining Power of Suppliers Few concentrated suppliers, high switching costs, specialized/differentiated inputs, no substitutes, ability to forward integrate Raises input costs
5. Rivalry Among Existing Competitors Many equal-sized competitors, undifferentiated products, slow growth, high exit barriers, history of price wars Directly depresses margins
All forces intense → no company earns attractive returns. Benign forces → highly profitable industry. Five Forces → assess long-run industry profitability (not just short-term).

Sustaining Innovation = improvement to existing product (led by incumbents). Disruptive Innovation = new market/value proposition initially for non-customers (usually by entrants). Innovator's Dilemma: incumbents slow to disrupt own profitable business.

Revenue Forecasting

  • Top-Down Drivers: GDP growth rate (relative), market size × market share
  • Bottom-Up Drivers: Volume × ASP; revenue by segment/product/geography; capacity-based; return-based
  • Recurring vs. Non-Recurring: Separate analysis; non-recurring excluded from base
  • Commodity Prices: Affect both revenue AND cost of sales simultaneously
Market Share Approach
Revenue = Market Size (addressable) × Market Share % Growth check: compare company growth vs market growth → gaining or losing share?
Capacity-Based
Revenue = Capacity × Utilization Rate × Price per Unit (common for airlines, hotels, utilities)

Operating Expense Forecasting

  • Short Horizon: Use drivers (variable costs % of revenue; fixed costs in $ terms)
  • Medium/Long Horizon: Use summary measures (margins) with industry mean reversion
  • CoGS: Forecast as % of revenue; commodity sensitivity analysis
  • SG&A: Largely fixed; may scale slightly with revenue
  • D&A: Based on prior gross fixed assets and useful lives
  • R&D: % of revenue or absolute; critical for pharma/tech
OpEx forecasts must be internally coherent with revenue forecast (e.g., if forecasting higher sales volume, variable costs must also rise).

Working Capital Forecasting

Standard Approach
AR (forecast) = DSO × (Revenue / 365) Inventory (forecast) = DOH × (CoGS / 365) AP (forecast) = DPO × (CoGS / 365) Use historical or industry DSO/DOH/DPO Adjust if business model changing
  • Mature company with unique model → use own historical ratios
  • New/changing company → use historical base rates / industry comps
  • Change in NWC = use of cash when NWC increases (current assets rise faster than liabilities)

Capex & Capital Structure Forecasting

  • Maintenance Capex – sustain current capacity; often ≈ D&A
  • Growth Capex – expand capacity; tied to expansion strategy and revenue growth targets
Capex Relationship
Total Capex = Maintenance Capex + Growth Capex Maintenance Capex ≈ Depreciation & Amortization (proxy) Growth Capex → linked to new store openings, capacity plans
  • Capital Structure Forecast – use target leverage ratios; management guidance; industry norms
  • Debt Issuance/Repayment – based on capex needs and FCF after dividends
  • Share Repurchases/Dividends – based on payout policy and excess FCF

Scenario Analysis & Risk Factors

  • Base Case – most likely outcome; single set of assumptions
  • Bull Case (Upside) – favorable outcomes realized
  • Bear Case (Downside) – adverse risks materialize
  • Analyst assigns probability to each scenario
  • Weighted average of scenario values → blended intrinsic value estimate

Three Categories of Equity Valuation Models

CategoryBasisModels
Present Value (DCF)PV of expected future cash flowsDDM, FCFE, FCFF models
Multiplier (Multiples)Price relative to fundamental variable; comparablesP/E, P/B, P/S, P/CF, EV/EBITDA
Asset-BasedMV of assets minus MV of liabilitiesAdjusted book value; liquidation value
Undervalued: Estimated Value > Market Price. Overvalued: Estimated Value < Market Price. Fairly Valued: Estimated Value ≈ Market Price. Analysts often use multiple models to triangulate value.

Dividend Types & Chronology

  • Regular Cash Dividend – paid periodically (quarterly US/Canada; semi-annual Europe/Japan)
  • Extra (Special) Dividend – one-off; cyclical companies or restructuring
  • Stock Dividend (Bonus Issue) – additional shares (2–10%); no economic effect; not cash
  • Stock Split – more shares, proportional price reduction; no economic effect
  • Reverse Stock Split – fewer shares, higher price; no economic effect
  • Share Repurchase (Buyback) – alternative to cash dividend; same wealth effect
Dividend Payment Timeline
Declaration Date → Ex-Dividend Date → Holder-of-Record Date → Payment Date (Buy BEFORE ex-date to receive dividend) Share price drops by ≈ dividend amount on ex-date
Reasons for Buybacks: (1) Signal shares undervalued (2) Flexibility (3) Tax efficiency if cap gains rate < dividend rate (4) Offset dilution from employee stock options

Dividend Discount Model (DDM) – All Formulas

General DDM
V₀ = Σ [Dt / (1+r)ᵗ] (t=1 to ∞) Dt = expected dividend at end of year t r = required rate of return
One-Period DDM
V₀ = (D₁ + P₁) / (1 + r)
P₁ = expected price at end of year 1
Gordon (Constant) Growth Model
V₀ = D₁ / (r − g) [if g constant forever] D₁ = D₀ × (1 + g) r > g required (else undefined) Where: D₁ = next period dividend r = required return on equity g = constant perpetual growth rate
⚠️ Use D₁ (NEXT year's dividend), NOT D₀
Sustainable Growth Rate (g)
g = b × ROE b = 1 − Dividend Payout Ratio = Retention Rate ROE = Net Income / Average Book Value of Equity
Two-Stage DDM
V₀ = Σ [Dt/(1+r)ᵗ] (t=1 to n) + Vn/(1+r)ⁿ Terminal Value: Vn = Dn+1 / (r − gL) Dn+1 = Dn × (1 + gL) Stage 1: Dividends grow at gS for n years Stage 2: Dividends grow at gL forever (gL < r)
Justified P/E (from Gordon Growth Model)
P/E = (D₁/E₁) / (r − g) = Payout Ratio / (r − g) P/E ↑ when: payout ↑, g ↑, r ↓ P/E ↓ when: r ↑, g ↓
Gordon model best for: mature, dividend-paying, relatively stable growth companies insensitive to business cycle (e.g., utilities, staples). NOT suitable for: non-dividend paying, high-growth, volatile.

Free Cash Flow to Equity (FCFE) Model

FCFE Formula
FCFE = CFO − FCInv + Net Borrowing CFO = Cash Flow from Operations FCInv = Fixed Capital Investment (≈ Capex net of disposals) Net Borrowing = New Debt − Debt Repayments
FCFE Valuation
V₀ = Σ [FCFEt / (1+r)ᵗ]
Preferred when: (1) Dividends differ significantly from dividend-paying capacity (2) Non-dividend-paying stock (3) Analyst takes control perspective
DDM vs. FCFE: DDM uses actual dividends. FCFE uses dividend-paying capacity. FCFE better captures true value when payout ≠ capacity.

Required Rate of Return – CAPM

Capital Asset Pricing Model (CAPM)
r = Rf + β × (Rm − Rf) Rf = Risk-free rate (e.g., 10-yr government bond yield) β = Beta (measure of systematic/non-diversifiable risk) Rm = Expected market return (Rm − Rf) = Equity Risk Premium (ERP) Required Return = Risk-Free Rate + β × ERP
Alternative methods: (1) Build-up: Rf + risk premium based on judgment (2) Bond yield + equity risk premium

Preferred Stock Valuation

Perpetual Non-Callable Preferred
V = D / r D = annual dividend; r = required rate of return Example: $5.50 dividend; r=6% → V = $5.50/0.06 = $91.67
Preferred Stock with Finite Maturity
V = Σ [D/(1+r)ᵗ] + F/(1+r)ⁿ (t=1 to n) F = par (face) value at maturity (Use periodic r and D matching payment frequency)
  • Callable preferred: lower value to investor (issuer calls when beneficial to issuer)
  • Putable (retractable) preferred: higher value (investor puts when beneficial)

Asset-Based Valuation

Asset-Based Equity Value
Equity Value = MV of Assets − MV of Liabilities − MV of Preferred Start with balance sheet; adjust book values to market values
  • Best suited: Companies with mainly tangible, marketable assets; financial companies; real estate; natural resources; liquidating companies
  • Limitation: Ignores intangibles (brand, goodwill, human capital, patents off-book) → provides "floor" value
  • Useful supplement to DCF/multiples; not primary method for most companies
  • Combination: Use asset-based + DDM + multiples for comprehensive view
