Reading 1
Rates and Returns
LOS 1.a: Interpret interest rates as required rates of return, discount rates, or opportunity costs; explain as sum of real risk-free rate + risk premiums.
Three Interpretations of Interest Rates
  • Required rate of return: Minimum return investors need to willingly lend funds.
  • Discount rate: Used to find PV of future cash flows.
  • Opportunity cost: Return forgone by consuming now instead of saving.
Key Formulas
Nominal risk-free rate ≈ Real risk-free rate + Expected inflation Required return = Real risk-free rate + Inflation premium + Default risk premium + Liquidity premium + Maturity premium
Risk Premiums (each raises required return)
PremiumCompensates for…
Default riskBorrower may not repay on time
Liquidity riskSelling below fair value if sold quickly
Maturity riskGreater price volatility of longer-term bonds
Real risk-free rate reflects time preference: degree to which current consumption is preferred to equal future consumption.
LOS 1.b: Calculate and interpret different approaches to return measurement.
Return Formulas
HPR = (P₁ − P₀ + D) / P₀ Arithmetic Mean = (R₁ + R₂ + … + Rn) / n Geometric Mean = [(1+R₁)(1+R₂)…(1+Rn)]^(1/n) − 1 Harmonic Mean = N / [Σ(1/Xᵢ)] (i = 1 to N)
Harmonic mean < Geometric mean < Arithmetic mean (when values are not all equal). Gap increases with dispersion.
When to Use Each Return Measure
MeasureUse Case
Arithmetic meanBest estimate of NEXT period's return; includes all values
Geometric meanCompound returns over MULTIPLE periods
Harmonic meanAverage cost per share with equal periodic $ investments (cost averaging)
Trimmed meanRemove top/bottom x% — reduces outlier effect
Winsorized meanReplace extremes with percentile values — reduces outlier effect
LOS 1.c: Compare money-weighted and time-weighted rates of return; evaluate portfolio performance.
Money-Weighted Return (MWR)
  • = IRR of portfolio cash flows.
  • Solve: CF₀ + CF₁/(1+r) + … + CFn/(1+r)ⁿ = 0
  • Beginning value & deposits = inflows; withdrawals & ending value = outflows.
  • Sensitive to TIMING of cash flows.
Time-Weighted Return (TWR)
Step 1: Calculate HPR for each sub-period Step 2: TWR = [(1+HPR₁)(1+HPR₂)…(1+HPRn)]^(1/n) − 1
MWR vs. TWR Comparison

MWR < TWR when…Funds added just BEFORE low-return period

LOS 1.d: Calculate and interpret annualized returns and continuously compounded returns.
Annualized HPR = (1 + HPR)^(365/days) − 1 Continuously compounded return (r_cc): r_cc = ln(1 + HPR) = ln(P_end / P_begin) PV with m compoundings/year: PV = FV / (1 + r/m)^(m×n) PV with continuous compounding: PV = FV × e^(−r×n)
Continuously compounded returns are additive across periods: r(0,T) = r(0,1) + r(1,2) + … This is a key advantage for multi-period analysis.
Greater compounding frequency → higher effective rate → higher FV and lower PV for the same stated rate.

Reading 2
The Time Value of Money in Finance
LOS 2.a: Calculate and interpret PV of fixed-income and equity instruments.
Core TVM Formulas
PV = FV / (1 + r)^n [discrete] PV = FV × e^(−r×n) [continuous] Perpetuity: PV = CF / r Annuity PMT: PMT = PV × [r / (1 − (1+r)^(−n))]
Fixed-Income Securities
  • Zero-coupon bond: PV = FV/(1+r)ⁿ. One cash flow at maturity.
  • Coupon bond: N=periods, I/Y=YTM, PMT=coupon, FV=par → CPT PV.
  • Perpetual bond: PV = Coupon / YTM
  • Amortizing bond: Each payment = principal + interest (level payments).
  • YTM > coupon rate → Bond at DISCOUNT (PV < par)
  • YTM < coupon rate → Bond at PREMIUM (PV > par)
Equity — Dividend Discount Models (DDMs)
Preferred Stock (perpetuity): P₀ = D / k_p Gordon Growth Model (constant growth): P₀ = D₁ / (k_e − g_c) where k_e > g_c Multistage DDM: Step 1: Discount dividends during high-growth period individually (as single CFs) Step 2: Apply constant growth DDM at end of high-growth period → get terminal P Step 3: Discount terminal P back to today
D₁ = next period's dividend = D₀ × (1+g). k_e = required return on equity. g_c = constant perpetual growth rate.
LOS 2.b: Calculate implied return and implied growth of equity instruments.
Implied required return (equity, constant growth): k_e = D₁/P₀ + g_c = Dividend yield + Growth rate Implied growth rate: g_c = k_e − D₁/P₀ = Required return − Dividend yield Implied YTM (bond): Given PV, FV, PMT, N → CPT I/Y
Price and yield are INVERSELY related — always. When yield rises, price falls; when yield falls, price rises. This is fundamental for all fixed-income analysis.
LOS 2.c: Explain cash flow additivity principle; implied forward rates, forward FX, option pricing.
Cash Flow Additivity / No-Arbitrage

PV of any stream = sum of PVs of individual cash flows. Identical future cash flows MUST have identical PVs (Law of One Price).

Forward Interest Rates
No-arbitrage: (1+S₂)² = (1+S₁)(1 + 1y1y) Solve for 1-year forward rate, 1 year from now: 1y1y = (1+S₂)² / (1+S₁) − 1 Notation: 1y1y = 1-yr loan, 1 yr from now 2y1y = 1-yr loan, 2 yrs from now
Forward Exchange Rates
F(price/base) = S(price/base) × (1 + r_price)/(1 + r_base) Higher interest rate currency trades at forward PREMIUM (in that currency's terms) to prevent arbitrage.
Binomial Option Pricing
Hedge ratio: h = (C_u − C_d) / (S_u − S_d) Portfolio value after 1 period is risk-free: V₀ = V₁ / (1 + r_f) c₀ = h×S₀ − V₀

Statistical Measures of Asset Returns
LOS 3.a: Calculate, interpret, and evaluate measures of central tendency and location.
Central Tendency Formulas
Sample Mean: X̄ = ΣXᵢ / n Median: Middle value (sorted); average of 2 middle values if n is even. Mode: Most frequent value; can be multimodal.
Outlier-Resistant Means
  • Trimmed mean: Remove top/bottom x% → compute mean. Fewer observations.
  • Winsorized mean: Replace extremes with boundary percentile values → compute mean. Same n.
Quantiles (Measures of Location)
QuantileDivides IntoKey Example
Quartile4 partsQ1=25th, Q2=50th (median), Q3=75th percentile
Quintile5 partsEach = 20th percentile
Decile10 partsEach = 10th percentile
Percentile100 partspth percentile = value below which p% of data falls

Interquartile Range (IQR) = Q3 − Q1 (middle 50% of data). Box-and-whisker plot: box = IQR; line in box = median.

LOS 3.b: Calculate, interpret, and evaluate measures of dispersion.
Dispersion Formulas
Range = Max − Min MAD = Σ|Xᵢ − X̄| / n Sample Variance: s² = Σ(Xᵢ − X̄)² / (n−1) [n−1 corrects bias vs population] Sample Std Dev: s = √[Σ(Xᵢ − X̄)² / (n−1)] CV (Coeff. of Variation) = s / X̄ [risk per unit of return; LOWER is better] Target Downside Deviation: = √[Σ(Xᵢ − R_T)² / (n−1)] for Xᵢ < R_T only [denominator still n−1, not count of below-target obs]
Interpreting Dispersion Measures
MeasureKey Point
RangeSimple; uses only 2 obs; very sensitive to outliers
MADAvg absolute deviation; more intuitive than variance
Variance / Std DevMost common risk measures; std dev in same units as data
CVCompare risk across assets with different mean returns
Target Downside DevDownside risk only; useful when investors care about losses below a specific target
Use sample variance (÷ n−1) when estimating population variance from a sample. Population variance uses ÷ n.

LOS 3.d: Interpret correlation between two variables.
Sample Covariance: Cov(X,Y) = Σ[(Xᵢ−X̄)(Yᵢ−Ȳ)] / (n−1) Correlation (Pearson): ρ_XY = Cov(X,Y) / (s_X × s_Y) Range: −1 ≤ ρ ≤ +1
ρ valueMeaning
+1.0Perfect positive linear relationship
−1.0Perfect negative linear relationship
0No LINEAR relationship (nonlinear may still exist)
  • Correlation has no units.
  • Correlation ≠ causation.
  • Spurious correlation: May arise by chance or from a third variable affecting both.
  • Outliers can inflate/deflate correlation — investigate if removing them significantly changes ρ.
  • Scatter plots reveal nonlinear relationships that ρ misses.
Scatter plots are the best way to detect nonlinear relationships where ρ ≈ 0 but a clear pattern exists.
Reading 4
Probability Trees and Conditional Expectations
LOS 4.a: Calculate expected values, variances, and standard deviations from probability models.
Expected Value: E(X) = Σ [P(xᵢ) × xᵢ] Variance: σ² = Σ [P(xᵢ) × (xᵢ − E(X))²] Std Dev: σ = √σ²
Unlike sample variance (÷ n−1), probability model variance uses probability weights — no n needed because the model defines the full population distribution.
Probability Tree Structure
  • Each branch shows a conditional probability.
  • Joint probability of a path = product of branch probabilities along that path.
  • All joint probabilities at terminal nodes must sum to 1.
  • E(X) = Σ (joint probability × outcome) across all terminal nodes.
LOS 4.b: Formulate an investment problem as a probability tree; explain conditional expectations.
  • Conditional expected value E(X|Y): Expected value of X given event Y has occurred. Used to revise expectations when new information arrives.
  • Conditional probability P(A|B) = P(AB)/P(B): Probability of A given B has occurred.
  • Probability trees make conditional expected values explicit and easy to compute.
LOS 4.c: Calculate and interpret an updated probability using Bayes' formula.
Bayes' Formula
P(A|B) = P(B|A) × P(A) / P(B) Total probability rule: P(B) = P(B|A)×P(A) + P(B|Aᶜ)×P(Aᶜ) Combined: P(A|B) = P(B|A)×P(A) / [P(B|A)×P(A) + P(B|Aᶜ)×P(Aᶜ)]
TermMeaning
P(A)Prior probability (before new information)
P(B|A)Likelihood — P of observing B if A is true
P(A|B)Posterior probability — updated P of A given B observed
Bayes process: (1) Draw probability tree, (2) Compute all joint probabilities, (3) Sum joints for the observed event B, (4) Divide P(B∩A) by P(B) = posterior P(A|B).
Reading 5
Portfolio Mathematics
LOS 5.a: Calculate expected value, variance, std dev, covariances, and correlations of portfolio returns.
Portfolio Expected Return: E(R_p) = Σ wᵢ × E(Rᵢ) where wᵢ = MV_i / Total MV 2-Asset Portfolio Variance: σ²_p = w²_A σ²_A + w²_B σ²_B + 2 w_A w_B Cov(R_A, R_B) Equivalent using correlation: σ²_p = w²_A σ²_A + w²_B σ²_B + 2 w_A w_B ρ_AB σ_A σ_B 3-Asset Portfolio Variance: σ²_p = w²_A σ²_A + w²_B σ²_B + w²_C σ²_C + 2w_A w_B Cov_AB + 2w_A w_C Cov_AC + 2w_B w_C Cov_BC
Covariance Matrix Facts
  • Diagonal elements = variances: Cov(Rᵢ, Rᵢ) = Var(Rᵢ)
  • For n assets: n variances + n(n−1)/2 unique covariances.
  • Cov(R_A, R_B) = Cov(R_B, R_A) — symmetric.
  • Lower covariance / correlation → lower portfolio variance → diversification benefit.
Correlation in a correlation matrix: diagonal values are always 1 (each asset perfectly correlated with itself).
LOS 5.b: Calculate covariance and correlation from a joint probability function.
Cov(R_A, R_B) = Σ P(sᵢ) × [R_A,i − E(R_A)] × [R_B,i − E(R_B)] Correlation: ρ_AB = Cov(R_A, R_B) / (σ_A × σ_B)
LOS 5.c: Define shortfall risk; calculate safety-first ratio; identify optimal portfolio using Roy's criterion.
Roy's Safety-First Criterion
Safety-First Ratio (SFR) = [E(R_p) − R_L] / σ_p where R_L = threshold (minimum acceptable) return Steps: 1. Calculate R_L = (Target ending value − Beginning value) / Beginning value 2. Compute SFR for each portfolio 3. Select portfolio with LARGEST SFR → Smallest probability of return < R_L
  • Shortfall risk: Probability that portfolio return falls below threshold R_L.
  • SFR = number of standard deviations the expected return is above the threshold.
  • Larger SFR → threshold farther from mean → smaller left-tail area → lower shortfall probability.
  • Shortfall probability = N(−SFR) from standard normal table.
SFR looks like Sharpe ratio with R_L replacing the risk-free rate. Maximize SFR to minimize shortfall probability.

LOS 6.b: Describe Monte Carlo simulation and its investment uses.
Monte Carlo Simulation — Steps
  1. Specify distributions and parameters for each risk factor.
  2. Randomly generate values for each risk factor.
  3. Value the security with a pricing model.
  4. Repeat many times (hundreds to tens of thousands).
  5. Use distribution of simulated values to estimate mean value and variance.
Applications
  • Value complex/path-dependent securities
  • Simulate trading strategy profits/losses
  • Calculate Value at Risk (VaR)
  • Pension fund asset/liability modeling
  • Value portfolios with non-normal returns
AdvantagesLimitations
Can test scenarios beyond historical data rangeComplex to implement
Handles many risk factorsOnly as good as assumptions (GIGO)
Can model any distribution shapeStatistical, not analytic — no closed-form insights

LOS 7.b: Explain the central limit theorem and its importance.
Central Limit Theorem (CLT)
  • For any population with mean μ and finite variance σ², the sampling distribution of the sample mean approaches normal as n increases.
  • Requires n ≥ 30 for the approximation to be reliable.
  • Mean of sampling distribution = μ (population mean).
  • Variance of sampling distribution = σ²/n.
Standard Error of Sample Mean
When σ is KNOWN: σ_x̄ = σ / √n When σ is UNKNOWN: s_x̄ = s / √n [use sample std dev s]
As n increases, SE decreases → sample mean converges to true mean. Doubling n reduces SE by factor √2 ≈ 1.41.
LOS 7.c: Describe the use of resampling (bootstrap, jackknife) to estimate the sampling distribution.

See Reading 6 LOS 6.c for detailed bootstrap/jackknife comparison. The CLT enables us to use z-tests and t-tests for inference regardless of population distribution when n ≥ 30. Resampling provides alternative SE estimates when analytical methods are difficult.

Key Rules for Each Test
  • t-test (mean): Use when σ unknown. Large n → t ≈ z.
  • Diff in means: Requires independent samples from normal populations. Pooled variance when equal variances assumed; separate variances otherwise.
  • Paired comparisons: Use when samples are DEPENDENT (e.g., same firms before/after event). Tests mean of differences ≠ 0.
  • Chi-square: Bounded at zero; two critical values for two-tailed test (upper and lower).
  • F-test: Always put LARGER variance in numerator → only need upper critical value. Reject H₀ if F > F_c.

Reading 9
Parametric and Nonparametric Tests of Independence
LOS 9.a: Parametric and nonparametric tests of the hypothesis that population correlation = 0.
Parametric — Pearson Correlation Test
H₀: ρ = 0 vs. Hₐ: ρ ≠ 0 t = r × √(n−2) / √(1−r²) df = n−2 Reject H₀ if |t| > t_critical (at α/2, df = n−2).
  • Test statistic increases with both higher |r| AND larger n.
  • Assumes both variables are normally distributed.
Nonparametric — Spearman Rank Correlation
r_s = 1 − [6 × Σdᵢ²] / [n(n²−1)] where dᵢ = difference in ranks for obs i (tied ranks: assign average rank) Test statistic (n > 30): t = r_s × √(n−2) / √(1−r_s²) ~ t(n−2)
Use Spearman when data are ranks (ordinal), or when the normality assumption of Pearson is not met.
LOS 9.b: Explain tests of independence based on contingency table data.
Chi-Square Contingency Table Test
H₀: Two characteristics are INDEPENDENT Hₐ: Two characteristics are NOT independent Test statistic: χ² = Σ (Oᵢⱼ − Eᵢⱼ)² / Eᵢⱼ for all cells (i,j) Expected frequency: Eᵢⱼ = (Row i total × Column j total) / Grand total Degrees of freedom: (r−1)(c−1) where r = number of rows, c = number of columns Reject H₀ if χ² > χ²_critical (at chosen α)
  • A contingency (two-way) table shows the number of observations with combinations of two characteristics.
  • Tests whether the two categorical characteristics are independent of each other.
  • Uses the chi-square distribution — same distribution as the variance test, but different purpose.
Don't confuse: Reading 8 chi-square tests a single variance. Reading 9 chi-square tests independence between two categorical variables from a contingency table.

Simple Linear Regression
LOS 10.a: Describe simple linear regression; least squares criterion; interpret coefficients.
Population model: Yᵢ = b₀ + b₁Xᵢ + εᵢ OLS estimated line: Ŷᵢ = b̂₀ + b̂₁Xᵢ Slope: b̂₁ = Cov(X,Y) / Var(X) Intercept: b̂₀ = Ȳ − b̂₁X̄ [line passes through (X̄, Ȳ)] OLS minimizes SSE = Σ(Yᵢ − Ŷᵢ)²
  • Dependent variable (Y): Explained / endogenous / predicted variable.
  • Independent variable (X): Explanatory / exogenous / predicting variable.
  • b̂₁: Expected change in Y per 1-unit change in X.
  • b̂₀: Expected value of Y when X = 0.
  • Beta: Slope of regressing security excess returns on market excess returns = systematic risk.

LOS 10.e: Calculate predicted value and prediction interval.
Predicted value: Ŷ_p = b̂₀ + b̂₁ × X_p Prediction interval: Ŷ_p ± t_c × s_f where df = n−2 s_f = standard error of the forecast (given on exam)
Prediction interval for a SPECIFIC value of Y is WIDER than a confidence interval for the mean of Y at a given X, because it includes individual (idiosyncratic) variation around the regression line.

ConceptDefinition

Unsupervised learningOnly input data (unlabeled); machine discovers structure and patterns.
